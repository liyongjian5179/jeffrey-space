# Responsive Checklist

Responsive behavior is a production requirement for Jeffrey Space, not a desktop layout squeezed onto smaller screens.

## Viewports

- Desktop: `1440 x 1100`
- Laptop: `1280 x 800`
- Tablet: `820 x 1180`
- Mobile: `390 x 900`
- Narrow mobile: `360 x 780`

## P0

- No horizontal scrolling on any viewport.
- Hero title does not overflow in Chinese or English.
- Hero IP illustration does not overpower the primary copy.
- CTA buttons are readable and easy to tap on mobile.
- Language switch is visible and usable on mobile.
- Belief slides do not overlap with the section title in Chinese or English.
- Project cards and modal galleries remain usable on mobile.
- Contact actions are clear, tappable, and do not wrap awkwardly.
- Footer ICP, public security filing, PV, and UV remain readable.

## P1

- Section spacing feels intentional on mobile.
- Card grids collapse cleanly to one column.
- Timeline cards read naturally in a single column.
- Profile avatar, stickers, and captions do not overlap.
- Photos preserve key subjects after cropping.
- Typography hierarchy stays distinct across languages.
- Online fonts have acceptable fallback behavior before loading.

## P2

- Tablet layout keeps a balanced rhythm between content and visuals.
- First viewport can be used as a clean sharing screenshot.
- Modals can be closed with pointer interaction and Escape.
- Reduced-motion preference keeps content visible without animation dependency.
- The site still feels like Jeffrey Space when external fonts are delayed.
